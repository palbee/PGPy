Objects¶

PGPy objects

PGPKeyring¶

class pgpy.PGPKeyring(keys=None)

PGPKeyring objects represent in-memory keyrings.

Parameters:keys (str, bytes, file-like object, list, None) – Accepts None, a path, URL, file-like object, or byte-string containing ASCII or binary encoded PGP/GPG keys. Can also accept a list of any of the above types, or combination therein.
key(id=None)

Context manager method. Select a key to use in the context managed block:

k = pgpy.PGPKeyring([os.environ['HOME'] + '/.gnupg/pubring.gpg', os.environ['HOME'] + '/.gnupg/secring.gpg'])
with k.key('DEADBEEF'):
    # do things with that key here
Parameters:id (str) – Specify a Key ID to use. This can be an 8 or 16 hex digit key ID, or a full key fingerprint, with or without spaces. Specifying no key (or None) is acceptable for key verification.
Raises:PGPError is raised if the key specified is not loaded.
load(keys)
Parameters:keys (str, bytes, file-like object, list) – Accepts a path, URL, file-like object, or byte-string containing ASCII or binary encoded PGP/GPG keys. Can also accept a list of any of the above types, or combination therein.
sign(self, subject)

Sign a document using the selected private key:

k = pgpy.PGPKeyring([os.environ['HOME'] + '/.gnupg/pubring.gpg', os.environ['HOME'] + '/.gnupg/secring.gpg'])
with k.key('DEADBEEF'):
    sig1 = k.sign('This is an unsigned document')
    sig2 = k.sign('path/to/another/unsigned/document')
Parameters:subject (str, bytes, file-like object) – Accepts a path, URL, file-like object, or byte-string containing the thing you would like to sign
Returns:newly created signature of the specified document.
Return type:PGPSignature
Raises:NotImplementedError if the selected key is not an RSA key
unlock(passphrase)

Decrypt encryted key material in a protected private key:

k = pgpy.PGPKeyring([os.environ['HOME'] + '/.gnupg/pubring.gpg', os.environ['HOME'] + '/.gnupg/secring.gpg'])
with k.key('DEADBEEF'):
    k.unlock('Dead Beef')
    # now that the private key is unlocked, use it for things
Parameters:passphrase (str) – The passphrase used to decrypt the encrypted private key material.
Raises:PGPError if the key specified is not encrypted
Raises:PGPKeyDecryptionError if the passphrase was incorrect
verify(subject, signature)

Verify the integrity of something using its signature and a loaded public key:

k = pgpy.PGPKeyring([os.environ['HOME'] + '/.gnupg/pubring.gpg', os.environ['HOME'] + '/.gnupg/secring.gpg'])
with k.key():
    sv = k.verify('path/to/an/unsigned/document', 'path/to/signature')
    if sv:
        print('Signature verified with {key}'.format(key=sv.key.keyid)
Parameters:
  • subject (str, bytes, file-like object) – Accepts a path, URL, file-like object, or byte-string containing the thing you would like to verify
  • signature (str, bytes, file-like object) – Accepts a path, URL, file-like object, or byte-string containing the signature of the thing you would like to verify
Returns:

indicating whether or not the signature verified.
Return type:

SignatureVerification¶

class pgpy.signature.SignatureVerification

Returned by pgpy.PGPKeyring.verify()

Can be compared directly as a boolean to determine whether or not the specified signature verified.

signature

The PGPSignature that was used in the verification that returned this

key

The PGPKey (if available) that was used to verify the signature

subject

The subject of the verification

class pgpy.pgp.PGPSignature

Returned by pgpy.PGPKeyring.sign()

path
Type:str, None

The local path of this signature, if set. Defaults to None for new signatures.
